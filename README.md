vagrant Cookbook
================
Installs/configures vagrant

Usage
-----
#### vagrant::default

Just include `vagrant` in your node's `run_list`:

```json
{
  "name":"my_node",
  "run_list": [
    "recipe[vagrant]"
  ]
}
```
